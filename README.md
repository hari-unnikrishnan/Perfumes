# GTG Perfumes Webpage

This is a responsive webpage built from a Figma design using HTML, CSS, and JavaScript.

## Setup Instructions

1. Replace the placeholder images in `assets/images/` with actual images exported from the Figma design:
   - `product1.jpg`, `product2.jpg`, `product3.jpg`: Product gallery images
   - `logo.png`: Company logo
   - `hero-bg.jpg`: Hero section background image

2. Open `index.html` in a web browser to view the page.

## Features

- Responsive design (desktop, tablet, mobile)
- Interactive product gallery with thumbnails and navigation
- Functional radio buttons for fragrance and purchase type selection
- Dynamic "Add to Cart" link based on selections
- Animated percentage counters
- Hamburger menu for mobile navigation

## Technologies Used

- HTML5
- CSS3 (with Grid and Flexbox)
- Vanilla JavaScript

## Browser Compatibility

- Chrome
- Firefox
- Edge
- Safari
